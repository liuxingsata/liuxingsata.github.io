成都记忆

发表于2021-09-30|更新于2021-09-30
|字数总计:3.7k|阅读时长:10分钟|阅读量:

没有照片，也可以有，等哪天有了心情，整理一下

1. 观察

成都第一天的生活，不紧不慢，不慌不忙，想走就走，想逛就逛，想吃就吃，想躺就躺

慢下来，才开始观察周围，观察周围人，也开始观察自己

  1. 这间民宿很优秀，处处透露出房东的细心和热情，让人感觉很舒服，格局装饰也非常温馨，很荣幸入住这间房子
  2. 出门就是宽窄巷子，逛了一圈，有一些宅院可以进去参观，算是有些味道，但总的来说商业化严重，东西贵且不好吃，不远处的奎星楼街上倒是有不少美食，有不少网红店，开始尝试第一波作死，尝试了冒椒火辣串串，emmmmm，很不错，除了里面的泡椒牛肉，是真的辣。。。
  3. 看了少年的你，讲的是个啥校园霸凌，不是说没有同情心，这种事有人关注自然是好事，但是相比之下，少年的那种泰然自若的视死如归，确实有打动到我，最后两人对视着，笑，哭，恍惚间，觉得我的生命里似乎缺少了什么
  4. 一直以来，我都是个以自我为中心的人，没有什么能够真正的影响到我，身边的人，身边的事，我都无感，工作也好，情感也罢，事情做便做了，喜欢一个人便喜欢了，好与坏似乎也不是很关心，有些人看了说，干的不错，值得嘉奖，咱也不知道哪里干的不错，那反正你都这么说了，我就姑且认同了，也正因为如此，我很少真正的顾及别人的感受，就算是我认为我喜欢的人，这大概也是没有几个知心朋友的原因吧，说起来，就算是他们几个，对我来说算是知心朋友么，朋友那一定是，甚至早已经超越了朋友，印在骨子里的甚至是一辈子的情感，但是说起知心，我自知没有认真考虑过他们的感受，而我自己也从未想过说出我的想法，或者说，我其实从来就没有想法，自己还觉得看透了一切一样，无所谓的态度，事实上这就是失败者的姿态，所谓社畜，不过如此
  5. 有那么一刻，突然脑子里闪过了我身边的这些人，从别人的角度出发去思考问题，可能很难，尤其是缺少人生阅历的情况下，但可以用心，不一定要切合需求，只要用心做好自己认为该做的事就好，在意的人会看到你的用心，不在意的人也会记住你的热心
2. 倾听

成都第二天是一场倾听的体验，用耳朵，用眼，用心；

司机是老成都人，船模比赛国家队员，儿子刚从中国传媒大学导演专业毕业，北京大学生街舞比赛冠军；船哥，他说这样称呼他，业余带带游客，故事也就从他的口中开始

  1. 熊猫是国宝，也成了成都的符号元素，憨态可掬，倒是没想过熊猫这么灵活，挂在树杈上睡觉，当然，稍年长些的还是如我等一般，开始懂得行为得体一些，不爬，不跑，就一个姿势，吃，完事睡，熊猫从小到大的一步步都呈现在眼前，我们看得到卖萌，看得到憨憨，开心的不得了，咱也不知道哪里值得开心，反正开心就对了，反正有些熊猫，奇一，龙龙，啥的，大概是有这两只熊猫吧，还有故事可以听，虽然讲解的不那么专业，但是听的还是津津有味，成都的熊猫吃竹子，也不是什么竹子都吃，他们更爱吃竹笋，但这个季节没有，就吃竹叶，抓着竹子，从一根竹叶开始，一根一根的用嘴的侧面捋着，直到差不多凑够一撮，丢掉竹子，熊猫掌握着这撮竹叶，开始一点点吃。。。emmmm,反正就可能吃得久了，总结出的经验，这样吃效率会高一点，不过也很浪费。。。其实，也差不多么，我们何尝不是如此
  2. 去过了798这种地方，体验过那种格格不入的感觉，不想强装文艺，但也确实，这种地方是一个消遣划水的好去处，至少比压马路好了一点，emmmmm，当然，对于情侣来说，我说的是成都的东郊记忆。。。这个地方，船哥倒是有不少故事，不过，我确实对那些年代感很强的故事不怎么感冒，也基本没记住。。。
  3. 消遣了一波川剧，近距离看看变脸，当然，不好奇，没啥兴趣，但是那一家三口还津津有味，我也看看。。。羡慕着他们
  4. 成都人的慢应该很大程度上体现在业余生活上，就比如人民公园里的这群人，年轻人不多，多是游客，密密麻麻，人山人海，像是什么集会，细看竟然是相亲大会，可真是闪了我的老腰，这一下确实让我感受到，我还年轻，嗯，还年轻，，鹤鸣茶社，据说范冰冰就坐在这湖边喝茶，嗯，成都的盖碗茶，嗑瓜子，打扑克，掏耳朵，就这样听故事，听了好久，好久。。。然后，天也没黑，我突然有点慌，时间怎么这么多。。。
  5. 时间本来不多，做的事情多了，时间就少了，但在成都，就感觉时间好多，是我做的事太少了吧，但其实，我做了很多的事，甚至比在北京的这些年都多，不过多解释，只是抒发想法；
  6. 听的故事好多，信息量好大，以至于我需要用第二天的上午来理清思路（当然就是太累想睡个懒觉），咱也不知道咋的就聊到了找对象的事情上，小夫妻相视一笑，讲了波他们的故事，emmmmm,反正就很温馨，重点是，我们的船哥，竟然讲起了他的一段算是佳话的故事，如何从遥远的东北牡丹江，结识了相守几十年的爱人，又是怎么不顾一切，带着爱人来到成都。。。从他豪爽的性格中，看得出一些当年的影子，咱听故事，听就完事了，到后面还总结了一句话，说追女孩就得胆子大。。。咱也不敢吱声，。。
3. 慵懒

成都的第三天，有些慵懒，走得少，思考的多

本来还有想去都江堰的计划，毕竟一大地标，怎么说也应该尽到做游客的本分，但咱一想，也不能完全按套路出牌啊，似乎一切都按照约定好的，走一遭，然后拍个照，打个卡，问题咱也不打卡啊，打卡给谁看啊。。。

  1. 这间民宿真的舒服，舒服的沙发，影院式的大屏幕，甚至还是vip，屋里有空调，还有一个贴心的小暖风机，对这间民宿说不出的感动，想着能给房东，给后来人留下一些礼物，和彩蛋，嘿嘿，上午就慵懒的不想起来，就这样享受这一份难得的体验。。。
  2. 我越来越觉得，我是做了一个多么正确的决定，生活就是需要这样的节奏，放缓自己的脚步，才知道自己究竟是在前进还是后退。。。。我说的是，一定要出去走走，走到一个完全不同的城市，的这个决定
  3. 美食当然不能放过，除了火锅，成都还是有很多知名小吃，美食决定了我的游玩路线，对于景点，顺路的话，可以去逛逛，比如下午去了锦里，emmmm，还象征性的掏个耳，emmmm，咱就玩么，不体验怎么能行。。。锦里，宽窄巷子，人民公园，诸如此类，为游客设立也好，成都人的生活方式也好，呈现的形象，复古，古色古香的建筑，随处可见，从传统美食，到手工艺品，到盖碗茶，扑面而来的就是这种复古感，似乎这是蜀地的元素之一；另一种元素就是熊猫，毫无疑问，在成都，不论哪里都有这个憨憨。。。迫不得已，我选择找一个猫友咖啡，平衡一下对萌与憨的理解。。。
  4. 我不擅长记录文字，只会写写心中所感，私以为此种思绪是生活中不可或缺，记录不一定要给人看，或许只是为了整理一下纷乱的无奈，但我很享受这种感觉，这间玉林西路的咖啡屋，二楼空间不大，刚好听得到楼下的旋律，又不吵闹，椅子上有两只猫窜过来，找个舒服的姿势开始，嗯，就趴在我的包上，开始睡。。。我还真就挺享受这感觉，不想装的很文艺，，，可偏偏就有点乐在其中
  5. 差不多该转场了，今天还有一波美食可以享用，还有一场喧嚣可以燥起来。。。
  6. 听司机说，老妈蹄花几经易手，早已失去了所谓的正宗，味道也变了，恰巧玉林路这边偶遇了一个小店，人还蛮多，点了一份蹄花，极其软糯，汤泡饭，肉沾着辣椒调的汁，算是吃的很满足的一餐
4. 疲惫

成都的第四天，真的是好累，后面甚至有种站都站不住的感觉，总想着找地方歇一歇，最后也没有找到。。。

  1. 陈麻婆豆腐出了名，11点才开门，去的时候刚刚营业，不需要排队，我倒是很喜欢店里的复古风格，每个区域像个小店，挂一个门牌，诸如药铺，饭铺此类；一个人，点了两个菜，一份小吃，钟水饺，夫妻肺片，陈麻婆豆腐（注册了商标，菜名就这么叫，其他店才叫麻婆豆腐），三份上来都是满满的红油，红色的辣椒，看着就贼有食欲。。。不过还好。。。这边这种红油其实并不是真的很辣（可能是为了应对游客，毕竟网红店），，，算是很有特色吧；
  2. 从成都博物馆里才算是了解了一些成都的兴衰史，风土人情，曾经的扬一益二辉煌时期，又经历过屠城，后来的民俗多来自于湖广填四川的人口迁移带来的，也正因这段历史，成都城内几乎没有保留下明朝以前的痕迹，今日的杜甫草堂等，全部都是人们按照史料，杜甫诗句，猜测假想建造，不过这倒无关紧要，毕竟就算是经历历史变迁，依然留存，又能有哪些影响呢。。。
  3. 春熙路商业街，远洋太古里，这种商业街其实不太适合我，我去了，尽到游客的本分，比预想的人多了一些，是看美女的好去处。。。呃呃呃，暴露了什么，不存在的。。。非要说有点印象的话，逛了一个猫空，随手寄了封信，给未来/；撞见了一间书店，方所，很有气息的书店，很多人，静静地坐在楼梯，过道，看书，这场面。。。书店挺大的，不只买书，有咖啡，有饰品，有周边，很全面，给我一种这才叫书店的感觉。。。不过此时逛的人已经站不稳了，没有久留
  4. 最后想吐槽的就是，我出来想找个地方吃一顿一个人的火锅，但是我还是小看了成都，小看了这里对于火锅的热情，一整条街的火锅店，无一例外要排很久的队，想着是吃不成了，还是拖着疲惫的身体，回去准备跨年吧。。。
5. 偶遇

成都的最后一天，偶遇了第一天的一家人，偶遇了青羊宫的腊八粥，偶遇了柯基咖啡店

  1. 最后一天跟着朱朱游览武侯祠，杜甫草堂，青羊宫，感受一波成都的文化底蕴，说起来朱朱还真是了解很多不为人知的故事，不论是真实故事，还是段子，似乎每一个地方，她都有话可讲，在杜甫草堂陪杜甫走完了潦倒的一生，不过朱朱站在一个女生的角度，高度赞扬了杜甫，一生守候在妻子身边。。。
  2. 武侯祠的故事其实主要是朱朱疯狂吐槽刘备，emmmm,反正我也不喜欢刘备，这个武侯祠是刘备的陵墓所在地，所谓君臣合祠，就有后人觉得诸葛亮和刘备是君臣楷模，硬给放到一起了，经历了不知道多少后人反复修缮，每个人都按照自己的意思折腾，emmmm,反正看着是那么回事就行了，听听别样的三国故事倒是也有些趣味
  3. 青羊宫算是少有的，道教的地盘吧，一般比较多的都是佛教寺院，成都的大慈寺也是地位不可撼动的存在，在这个道观里竟然也供奉些佛教的东西，还美其名曰，佛道结合，优秀的一批，既不信佛，也不悟道。。。出来时候偶遇门口施粥（因为刚好腊八，所以其实青羊宫的活动，是腊八粥）恰好就排到了我们，恰好就是最后几碗。。。味道极好的八宝粥。。。感觉整个人都温暖起来了，本来天气还下了点毛毛雨，有些阴冷。。。这种出其不意的温暖最是深刻
  4. 告别朱朱，时间还早，去找了那间无早，然而有些失望，就漫无目的的在街上闲逛了一会儿，一抬头竟然偶遇一间咖啡店，还是一个非常独特的宠物咖啡店，有柴犬，柯基，甚至还有两只浣熊。。。找了个角落，看着大厅里那么多狗狗，和那群人嬉闹，倒是有趣，咖啡店还可以这样开。。。今天意料之外的惊喜真的是。。。说起来我给爸妈发了个100块的拼手气红包，一共三个，我自己抢了1毛3分，真的是闻所未闻。。。

旅途结束的心满意足，成都的生活惬意，悠闲，有美食，有美女，呃呃呃，反正就是一个热情，温暖的城市，当然，不是晒太阳的那种温暖，毕竟成都哪里有太阳啊。。。
